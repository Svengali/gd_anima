# Anima.Node

This class is used to animate a single node.

## Syntax

```gdscript
Anima.Node(node: Node) -> AnimaDeclaration
```

| param | type | Description |
|---|---|---|
| node | Node | The node to animate |

## Example

```gdscript
Anima.begin_single_shot(self) \
    .with(
        Anima.Node($Label).anima_animation_frames({
            0: {
                "translate:x": "-:size:x",
            },
            100: {
                "translate:x": 0,
            }
        })
    ) \
    .play_with_delay(0.5)
```
